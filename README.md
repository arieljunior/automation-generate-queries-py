
# Queries Generate (Beta)

Generate queries to insert table with rules


## Install Python



```bash
  Python 3.11
```
    
## Run Local

Clone the project

```bash
  git clone https://github.com/arieljunior/automation-generate-queries-py.git
```

Entry directory

```bash
  cd my-project
```

Install dependencies

```bash
  pip install pyperclip
```

Start

```bash
  python main.py
```
